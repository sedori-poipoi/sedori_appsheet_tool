# AppSheet連携ツール 設定ガイド

このツールは、AppSheetでスキャンしたバーコードを元に、せどりポイポイの計算機能を使って自動で利益計算を行うためのものです。

## 構成
- `Code.gs`: Google Apps Script (GAS) 用のコード
   - AppSheetの裏側で動き、バーコードが入力されたら自動でAPIを叩いて情報を取得します。

## 設定手順

### 1. スプレッドシートの準備
1.  新しいスプレッドシートを作成します（または既存のものを使用）。
2.  シート名を「シート1」にするか、`Code.gs` 内の `TARGET_SHEET_NAME` を変更してください。
3.  1行目にヘッダーを作成します。
    - A列: ID (AppSheet用)
    - B列: バーコード (スキャン用)
    - C列: 商品名
    - D列: 粗利
    - E列: ランキング
    - F列: 詳細リンク
    - G列: 判定

### 2. GASの設定
1.  スプレッドシートのメニューから「拡張機能」>「Apps Script」を開きます。
2.  元々ある `myFunction` 等を消し、`Code.gs` の内容を全て貼り付けます。
3.  **重要**: `Code.gs` 内の `API_URL` を、あなたのせどりポイポイのデプロイ先URLに変更してください。
    - 例: `const API_URL = "https://your-app-name.vercel.app/api/appsheet/scan";`
4.  保存ボタン（フロッピーアイコン）を押します。

### 3. トリガーの設定（自動実行）
1.  左メニューの「トリガー（時計アイコン）」をクリックします。
2.  「トリガーを追加」ボタンを押します。
3.  以下のように設定します。
    - 実行する関数: `onEdit`
    - イベントのソース: `スプレッドシート`
    - イベントの種類: `編集時`
4.  「保存」を押します（認証画面が出たら「詳細」>「安全ではないページに移動」で許可してください）。

### 4. AppSheetの設定
1.  AppSheetでこのスプレッドシートを読み込みます。
2.  「バーコード」列のスキャン機能を有効にします。
3.  「詳細リンク」列に対して、Actionを設定します。
    - Do this: `External: Go to a website`
    - Target: `[詳細リンク]` （または `CONCATENATE(...)` でURL生成）

これで、スマホでスキャンするたびに、数秒後に自動で利益や判定が表示されるようになります！
